# Getting Started

1. Sign up for a free account at [proposales.com](https://proposales.com)

2. Find your API key at the bottom of: [https://secure.proposales.com/settings/profile](https://secure.proposales.com/settings/profile)

3. Explore our API docs: [https://docs.proposales.com](https://docs.proposales.com)

4. OpenAPI spec: [https://docs.proposales.com/api-reference/openapi.json](https://docs.proposales.com/api-reference/openapi.json)

> **Note**: Your account will start with an empty content library. Before your system can match products, you'll need to populate it with sample hotel products — meeting rooms, catering packages, AV equipment, accommodation options, event spaces, etc. Use the API to do this programmatically (see the Content endpoints in the docs). The more realistic your content library, the better your system can demonstrate its retrieval and matching capabilities.
